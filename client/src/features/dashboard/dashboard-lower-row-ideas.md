# Dashboard Lower Row Ideas

## Implemented

- Weekly Snapshot
  - Shows rides completed in the current week.
  - Includes distance, duration, and elevation totals.
  - Uses existing history data from dashboard mapper.

## Backlog Ideas

- Consistency / Streak Card
  - Current ride streak, longest streak, and next required ride day.
  - Helps users maintain momentum.

- Weather + Best Time Window
  - Next 2-3 ride-friendly windows using local weather.
  - Could include wind, temperature, and precipitation cues.

- Goals Tracker
  - Monthly distance and elevation goals with progress bars.
  - Optional "on track" indicator based on current date.

- Suggested Ride
  - One personalized route suggestion based on recent ride behavior.
  - Could prioritize typical distance, terrain, and preferred ride type.

- Social Pulse
  - Nearby club/friend activity summary for the coming weekend.
  - Fast social discovery without opening rides or clubs pages.
